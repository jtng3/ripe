# Ripe

**The fruit on your street, before it falls.**

A dead-simple neighborhood map for sharing surplus garden produce. Browse what's
ripe near you, claim a bag, and pay the grower directly — Ripe never touches the
money, so 100% stays on the block.

**Live:** https://ripe-psi.vercel.app

## Run locally

```bash
pnpm install
pnpm dev
# → http://localhost:3000
```

## Stack

- [Next.js 16](https://nextjs.org) · React 19 (App Router)
- Tailwind CSS v4 · [Motion](https://motion.dev)
- TypeScript · pnpm
